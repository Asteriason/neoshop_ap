Descripción del Proyecto:Mi proyecto es un bot de automatización para Neopets que busca y cotiza artículos de acuerdo con el precio más bajo en el asistente de búsqueda. Es un script de Python que utiliza la biblioteca Selenium WebDriver para navegar por el sitio web de Neopets, ingresar consultas de búsqueda y extraer detalles de elementos. El bot también puede iniciar sesión en una cuenta de usuario, manejar desafíos de CAPTCHA y ordenar los resultados de búsqueda por precio o rareza.Caso de negocio:El sitio web de Neopets es un popular juego en línea que permite a los usuarios crear y cuidar mascotas virtuales. Un aspecto del juego es la capacidad de comprar y vender artículos en el mercado de Neopian. Sin embargo, buscar y cotizar artículos puede llevar mucho tiempo y ser tedioso, especialmente para artículos raros o costosos. Al automatizar este proceso, mi bot puede ayudar a los usuarios de Neopets a ahorrar tiempo, encontrar mejores ofertas y concentrarse en otros aspectos del juego.Habilidades técnicas:Para construir mi bot de automatización, utilicé varias habilidades técnicas, que incluyen:• Programación en Python: Escribí el script en Python, un lenguaje de programación popular que es fácil de aprender y usar.• Selenium WebDriver: utilicé la biblioteca Selenium WebDriver para interactuar con el sitio web de Neopets, simular las acciones del usuario y extraer datos de las páginas web.• Web scraping: utilicé técnicas de web scraping para extraer nombres, descripciones, imágenes y precios de artículos de los resultados de búsqueda y las páginas de artículos.• Integración de API: utilicé la API de Neopets para autenticar la cuenta de usuario y acceder a ciertas funciones, como el inventario y la tienda del usuario.Resultados de la prueba:Para probar el bot, creé un plan de prueba que cubría los siguientes aspectos:• Funcionalidad: probé la capacidad del bot para buscar y poner precio a artículos en el mercado, iniciar sesión en una cuenta de usuario y manejar errores y excepciones.• Rendimiento: probé la velocidad y la eficiencia del bot para completar tareas, como buscar un artículo específico o fijar el precio de varios artículos a la vez.• Confiabilidad: probé la capacidad del bot para manejar situaciones inesperadas, como errores de red, cambios en el sitio web o desafíos de CAPTCHA.• Seguridad: probé el cumplimiento del bot con los términos de servicio y la política de privacidad de Neopets, y me aseguré de que no violaba ninguna regla o regulación.Según mis pruebas, descubrí que el bot funcionó bien en todos los aspectos y pudo mejorar la experiencia del usuario y ahorrar tiempo a los usuarios de Neopets.Calidad del código:Para garantizar la calidad del código de mi bot, seguí estas prácticas recomendadas:• Modularidad: Estructuré el código en funciones y módulos separados, haciéndolo fácil de leer y mantener.• Reusabilidad: Usé variables y parámetros para hacer el código adaptable a diferentes situaciones y entradas.• Legibilidad: utilicé nombres de variables significativos, comentarios y cadenas de documentos para que el código fuera fácil de entender.• Documentación: Escribí un archivo LÉAME que explicaba la instalación, el uso y la resolución de problemas del bot, y proporcionaba ejemplos y capturas de pantalla.• Control de versiones: utilicé GitHub para almacenar el código y realizar un seguimiento de los cambios, lo que facilitó la colaboración con otros desarrolladores y el intercambio de código con la comunidad.Mejoras futuras:En el futuro, planeo agregar las siguientes características a mi bot:• Opciones de búsqueda avanzada: permite a los usuarios filtrar los resultados de la búsqueda por tipo de artículo, rareza, rango de precios y nombre del vendedor.• Seguimiento de precios: permite a los usuarios realizar un seguimiento del precio de un artículo específico a lo largo del tiempo y recibir notificaciones cuando alcanza un cierto umbral.• Auto-compra: permite a los usuarios comprar automáticamente un artículo cuando alcanza un cierto precio o nivel de rareza.• Al implementar estas mejoras, mi bot puede convertirse en una herramienta aún más poderosa y conveniente para que los usuarios de Neopets administren sus activos virtuales y logren sus objetivos de juego.
